# Memory

## Overview

Memory is the ability to encode, store, and retrieve information. Without memory, we would have no sense of personal history, couldn't learn from experience, and couldn't function in daily life. Cognitive neuroscience has revealed that memory isn't a single system but multiple systems with distinct neural substrates and operating principles.

## Key Concepts

### Memory Systems Taxonomy

```
MEMORY
├── SHORT-TERM / WORKING MEMORY (seconds-minutes)
│   ├── Phonological loop
│   ├── Visuospatial sketchpad
│   └── Central executive
│
└── LONG-TERM MEMORY (minutes-lifetime)
    ├── EXPLICIT (Declarative) - conscious
    │   ├── Episodic (events, experiences)
    │   └── Semantic (facts, knowledge)
    │
    └── IMPLICIT (Non-declarative) - unconscious
        ├── Procedural (skills, habits)
        ├── Priming (perceptual facilitation)
        ├── Classical conditioning
        └── Non-associative (habituation, sensitization)
```

### Working Memory

**Definition:** System for temporarily holding and manipulating information

**Baddeley's Model:**

| Component | Function | Neural Basis |
|-----------|----------|--------------|
| **Phonological loop** | Verbal/acoustic info; subvocal rehearsal | Left inferior parietal, Broca's area |
| **Visuospatial sketchpad** | Visual/spatial info | Right hemisphere, parietal |
| **Central executive** | Attention control, coordination | Prefrontal cortex |
| **Episodic buffer** | Integrates info into episodes | Prefrontal, hippocampus |

**Capacity:** ~4 items (or chunks); highly limited

**Prefrontal Cortex Role:**
- Maintains information through sustained firing
- Dorsolateral PFC: manipulation, monitoring
- Ventrolateral PFC: maintenance, retrieval

**Working Memory vs. Short-Term Memory:**
- WM emphasizes manipulation and processing
- STM emphasizes simple storage
- WM is "STM + central executive"

### Long-Term Memory: Encoding

**Levels of Processing (Craik & Lockhart):**
- **Shallow** - Physical features (font, case) → poor memory
- **Deep** - Semantic meaning → better memory
- Elaborative encoding creates richer memory traces

**Encoding Specificity:**
- Memory best when retrieval context matches encoding context
- State-dependent learning (mood, environment)

**Schemas:**
- Knowledge structures that organize information
- Aid encoding and retrieval
- But can cause distortions (filling in schema-consistent details)

**Spacing Effect:**
- Distributed practice > massed practice
- Multiple sessions better than cramming

**Testing Effect:**
- Retrieval practice strengthens memory more than restudying
- Tests are learning events, not just assessments

### Long-Term Memory: Consolidation

**Definition:** Process by which memories become stable over time

**Two stages:**

**Synaptic consolidation (minutes-hours):**
- Cellular/molecular changes at synapses
- Protein synthesis required
- LTP mechanisms (NMDA receptors, gene expression)

**Systems consolidation (days-years):**
- Transfer from hippocampus to neocortex
- Hippocampus initially required; neocortex holds long-term
- **Standard model:** Complete transfer over time
- **Multiple trace theory:** Hippocampus always involved for episodic detail

**Sleep and Consolidation:**
- Memory replay during sleep
- Slow-wave sleep: declarative memory
- REM sleep: procedural memory
- Sleep deprivation impairs consolidation

### Long-Term Memory: Retrieval

**Recall vs. Recognition:**
- **Recall** - Generate memory from cue (harder)
- **Recognition** - Identify previously encountered item (easier)
- Different neural signatures

**Retrieval Cues:**
- External (environmental) or internal (mood, thoughts)
- Effective cues overlap with encoding

**Reconstructive Memory:**
- Memory is not replay of recording
- Active reconstruction based on fragments + inference
- Subject to distortion, false memories

**Retrieval-Induced Forgetting:**
- Retrieving some items can inhibit related items
- Competition during retrieval

### The Hippocampus and Medial Temporal Lobe

**MTL Structures:**
- Hippocampus (CA1, CA3, dentate gyrus)
- Entorhinal cortex
- Perirhinal cortex
- Parahippocampal cortex

**Hippocampal Functions:**

| Function | Evidence |
|----------|----------|
| **Episodic memory encoding** | Damage → anterograde amnesia |
| **Spatial navigation** | Place cells, cognitive maps |
| **Relational binding** | Connects elements into episodes |
| **Pattern separation** | Distinguishes similar memories |
| **Pattern completion** | Retrieves whole from partial cue |

**Patient H.M. (Henry Molaison):**
- Bilateral MTL removal for epilepsy
- Severe anterograde amnesia (couldn't form new episodic memories)
- Retrograde amnesia (few years before surgery)
- Intact working memory, procedural learning
- Landmark case demonstrating hippocampal role

### Episodic vs. Semantic Memory

| Feature | Episodic | Semantic |
|---------|----------|----------|
| Content | Personal events | Facts, concepts |
| Context | Time and place specific | Context-independent |
| Experience | "Remember" | "Know" |
| Neural basis | Hippocampus critical | Can survive MTL damage |
| Example | "My first day of college" | "The capital of France is Paris" |

**Interaction:**
- Semantic knowledge derived from episodic experiences
- Episodic retrieval uses semantic framework

### Implicit Memory Systems

#### Procedural Memory
- Skills and habits
- "Knowing how" vs. "knowing that"
- **Neural basis:** Basal ganglia, cerebellum, motor cortex
- Intact in amnesia (H.M. learned mirror tracing)

#### Priming
- Facilitated processing of previously encountered stimuli
- Can be:
  - **Perceptual** - Same physical features
  - **Conceptual** - Same meaning
- **Neural basis:** Reduced activity in relevant cortical areas (repetition suppression)
- Intact in amnesia

#### Classical Conditioning
- **Emotional conditioning:** Amygdala
- **Eyeblink conditioning:** Cerebellum
- Can occur without awareness

### Memory Failures and Distortions

**Schacter's "Seven Sins of Memory":**

| Sin | Type | Description |
|-----|------|-------------|
| **Transience** | Forgetting | Memory fades over time |
| **Absent-mindedness** | Forgetting | Inattention at encoding/retrieval |
| **Blocking** | Forgetting | Temporary retrieval failure (tip-of-tongue) |
| **Misattribution** | Distortion | Wrong source attribution |
| **Suggestibility** | Distortion | Incorporating misleading information |
| **Bias** | Distortion | Current knowledge distorts past memories |
| **Persistence** | Intrusion | Unwanted memories (trauma) |

**False Memories:**
- Can be vivid and confidently held
- DRM paradigm: Falsely "remember" related but non-presented words
- Misinformation effect: Post-event info alters memory
- Implanted memories possible through suggestion

### Memory and Emotion

**Enhanced emotional memory:**
- Emotional events remembered better
- Amygdala modulates hippocampal consolidation
- Stress hormones (cortisol, adrenaline) enhance consolidation

**Flashbulb memories:**
- Vivid memories for emotionally significant events
- Feel accurate but subject to distortion
- "9/11 memories" - confident but often inaccurate

**PTSD:**
- Intrusive, persistent traumatic memories
- Possible reconsolidation-based treatments

### Memory Across the Lifespan

**Infantile amnesia:**
- Few memories before age 3-4
- Hippocampus still developing
- Lack of mature self-concept

**Aging effects:**
- Episodic memory declines
- Semantic memory relatively preserved
- Working memory capacity decreases
- Slowed processing, retrieval difficulties

**Alzheimer's disease:**
- Progressive memory loss
- Hippocampal atrophy early
- Amyloid plaques, neurofibrillary tangles
- Eventually affects all memory systems

## Real-World Examples

- **Patient H.M.:** Bilateral MTL removal → couldn't form new memories; intact procedural learning
- **Clive Wearing:** Severe amnesia from encephalitis; 30-second memory, but can still play piano
- **London taxi drivers:** Enlarged posterior hippocampus from spatial navigation learning
- **Eyewitness testimony:** Highly unreliable due to reconstructive nature and suggestibility
- **Spaced repetition apps:** Apply spacing effect for efficient learning

## Key Terms

| Term | Definition |
|------|------------|
| **Working memory** | Temporary storage and manipulation of information |
| **Episodic memory** | Memory for personal events and experiences |
| **Semantic memory** | Memory for facts and general knowledge |
| **Procedural memory** | Memory for skills and habits |
| **Consolidation** | Process stabilizing memories over time |
| **Hippocampus** | MTL structure critical for episodic memory |
| **Anterograde amnesia** | Inability to form new memories |
| **Retrograde amnesia** | Loss of memories from before injury |
| **Priming** | Facilitated processing from prior exposure |

## Summary

Memory comprises multiple systems with distinct neural substrates. Working memory (prefrontal cortex) holds information temporarily. Long-term declarative memory depends on the hippocampus for encoding and consolidation, while implicit memories (procedures, priming) rely on basal ganglia, cerebellum, and sensory cortex. Memory is reconstructive, not reproductive—subject to distortion and false memories. Understanding these systems has profound implications for education, eyewitness testimony, and treating memory disorders.
