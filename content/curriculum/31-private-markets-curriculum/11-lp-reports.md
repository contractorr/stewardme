## 11.2 Quarterly Report Structure

### Typical Sections

```
QUARTERLY REPORT CONTENTS
─────────────────────────

1. GP LETTER / EXECUTIVE SUMMARY
   • Market commentary
   • Fund highlights
   • Key developments

2. FUND SUMMARY
   • Performance metrics (IRR, TVPI, DPI)
   • Capital deployment status
   • Commitment summary

3. PORTFOLIO OVERVIEW
   • List of investments
   • Status of each (active, realized, written off)
   • Valuation summary

4. PORTFOLIO COMPANY UPDATES
   • Individual company performance
   • Financial metrics
   • Strategic developments

5. CAPITAL ACTIVITY
   • Capital calls in period
   • Distributions in period
   • Upcoming calls (if known)

6. FINANCIAL SUMMARY
   • Condensed financials
   • Fee disclosure
   • NAV bridge

APPENDICES
├── Capital Account Statement (LP-specific)
├── Schedule of Investments
├── Cash Flow Summary
└── Benchmarking Data
```

### Reading the GP Letter

```
GP LETTER ANALYSIS
──────────────────

WHAT TO LOOK FOR:

MARKET COMMENTARY
├── GP's view on current environment
├── How it affects portfolio
└── Watch for: Excuses vs insights

PORTFOLIO HIGHLIGHTS
├── Exits and new investments
├── Notable developments
└── Watch for: What's NOT mentioned

PERFORMANCE DISCUSSION
├── Attribution of returns
├── Drivers of value creation
└── Watch for: Blame shifting on underperformance

FORWARD OUTLOOK
├── Exit pipeline
├── Deployment expectations
└── Watch for: Overly optimistic projections

RED FLAGS IN GP LETTERS:
⚠ Vague language about struggling investments
⚠ Excessive market blame for underperformance
⚠ Missing discussion of problem companies
⚠ Changing narrative quarter to quarter
⚠ Delays in reporting
```

```
PERFORMANCE ANALYSIS STEPS
──────────────────────────

1. ASSESS ABSOLUTE PERFORMANCE
   IRR: 15.8% net
   └── Is this good for the strategy?
   └── Buyout typically targets 15-20%
   └── This is solid

2. COMPARE TO BENCHMARK
   Fund: 15.8% vs Benchmark: 14.2%
   └── Outperforming by 1.6%
   └── First quartile
   └── Good relative performance

3. CHECK REALIZATION
   DPI: 0.31x (19% of TVPI)
   └── Fund is 5 years old
   └── Expected: 20-35% realized
   └── On track

4. ASSESS NAV RELIABILITY
   RVPI: 1.31x (81% of value unrealized)
   └── Significant NAV
   └── Returns could change
   └── Watch exit execution

5. TREND ANALYSIS
   Compare to prior quarters:
   └── IRR improving? ✓
   └── DPI growing? ✓
   └── RVPI stable? ✓
   └── Healthy progression
```

### Analyzing Your Capital Account

```
CAPITAL ACCOUNT ANALYSIS
────────────────────────

1. VERIFY THE MATH
   Closing NAV = Opening + Contributions - Distributions
                 + Income - Expenses + Gains - Carry

   $43,000,000 = $42,500,000 + $0 - $3,050,000
                 + $120,000 - $197,000 + $3,290,000 - $663,000
   $43,000,000 = $43,000,000 ✓

2. CHECK TVPI
   TVPI = (Distributions + NAV) / Contributions
   TVPI = ($15,300,000 + $43,000,000) / $45,250,000
   TVPI = 1.29x...

   Wait, report says 1.62x. Why?

   Answer: TVPI uses called capital for investments only,
   excluding fees. Called for investments ≈ $36,000,000
   TVPI = $58,300,000 / $36,000,000 = 1.62x ✓

3. TRACK CARRY ACCRUAL
   Opening carry accrual: ~$3,422,000
   This quarter accrual: $663,000
   Closing carry accrual: $4,085,000

   Carry is growing = NAV increasing = good sign

4. MONITOR REMAINING OBLIGATION
   Unfunded: $4,750,000
   Recallable: $2,000,000
   Total potential calls: $6,750,000

   Be prepared for future calls
```

### Red Flags in Portfolio Updates

```
WARNING SIGNS TO WATCH
──────────────────────

FINANCIAL CONCERNS:
⚠ Revenue declining or flat
⚠ Margins compressing
⚠ Missing projections
⚠ Cash burn accelerating
⚠ Covenant breaches mentioned

STRATEGIC CONCERNS:
⚠ Management turnover
⚠ Lost key customers
⚠ Competitive pressure increasing
⚠ Market conditions deteriorating
⚠ Delayed product launches

VALUATION CONCERNS:
⚠ Multiple expansion without justification
⚠ "Strategic value" without numbers
⚠ Peer multiples not disclosed
⚠ No mention of exit timeline
⚠ Down round by competitors

DISCLOSURE CONCERNS:
⚠ Less detail than prior quarters
⚠ Metrics no longer disclosed
⚠ Vague language
⚠ Excuses dominate commentary
⚠ Company missing from update entirely
```

## 11.7 Annual Report & Audited Financials

### Annual Report Additions

```
ANNUAL REPORT ADDITIONAL CONTENT
────────────────────────────────

BEYOND QUARTERLY:

1. FULL-YEAR SUMMARY
   • Complete year performance
   • Year-over-year comparison
   • Annual attribution analysis

2. AUDITED FINANCIAL STATEMENTS
   • Balance Sheet
   • Statement of Operations
   • Statement of Cash Flows
   • Statement of Changes in Partners' Capital
   • Notes to Financial Statements

3. AUDITOR'S OPINION
   • Unqualified = Clean (good)
   • Qualified = Issues noted (concerning)
   • Adverse = Material misstatement (bad)

4. FEE DISCLOSURE
   • Full fee transparency
   • Related party transactions
   • Portfolio company fees

5. VALUATION DETAILS
   • Level 3 assumptions
   • Valuation methodology by investment
   • Sensitivity analysis
```

## 11.8 Putting It All Together

### Comprehensive Fund Review

```
QUARTERLY REVIEW PROCESS
────────────────────────

STEP 1: HIGH-LEVEL SCAN (5 minutes)
├── Read GP letter
├── Check IRR/TVPI vs prior quarter
├── Note any material events
└── Flag items for deeper review

STEP 2: PERFORMANCE ANALYSIS (10 minutes)
├── Calculate changes from prior period

├── Assess realization progress
```

## 11.9 Common Questions & Issues

### Questions to Ask GP

```
CONSTRUCTIVE GP QUESTIONS
─────────────────────────

PERFORMANCE:
• What's driving the change in IRR this quarter?
• How do recent exits compare to last marks?
• What's the outlook for Portfolio Company X?

PORTFOLIO:
• What's the exit timeline for mature holdings?
• How is Company Y addressing [specific issue]?
• Are there any covenant concerns in the portfolio?

CAPITAL:
• What's the expected remaining capital calls?
• Will there be any distributions in near term?
• Is the subscription line currently drawn?

VALUATION:
• What drove the mark-up/down for Company Z?
• How did you select the comparable companies?
• What would change your valuation assumptions?
```

### Common Data Issues

```
DATA QUALITY ISSUES
───────────────────

COMMON PROBLEMS:
├── IRR calculated differently than expected
├── TVPI denominator inconsistency
├── Capital account doesn't reconcile
├── Missing prior period comparisons
├── Valuation timing mismatches

RESOLUTION STEPS:
1. Document the discrepancy
2. Check report notes/definitions
3. Compare to audited financials
4. Contact GP investor relations
5. Document resolution for future reference
```

## Knowledge Check

1. What's the typical timeline for receiving a quarterly report after quarter-end?
2. Why might the TVPI you calculate not match the reported TVPI?
3. What are three red flags to look for in portfolio company updates?
4. What does an "unqualified" auditor opinion mean?
5. How can you assess if a fund's valuations are reliable?

<details>
<summary>Answers</summary>

1. Typically 45-60 days after quarter end. Some funds provide preliminary estimates at 15-30 days.
2. TVPI denominator may differ - some use total contributions (including fees), others use only investment capital. Check the fund's definition.
3. Any three: declining revenue, management turnover, margin compression, missing metrics that were previously disclosed, vague language replacing specific data, competitive pressure, covenant issues.
4. An unqualified (clean) opinion means the auditor believes the financials fairly represent the fund's position with no material issues.
5. Compare exit proceeds to prior NAV marks (exits should be near or above last mark), check if write-downs occur when appropriate, see if multiples align with market comparables.

</details>
