## Prerequisites

- Basic understanding of financial concepts (revenue, profit, assets, liabilities)
- Familiarity with basic accounting (balance sheet, income statement)
- No prior private equity experience required

## How to Use This Curriculum

1. **Sequential learning**: Modules build on each other - complete in order
2. **Practice exercises**: Each module includes exercises to reinforce concepts
3. **Real-world examples**: Numerical examples use realistic fund scenarios
4. **Cross-references**: Links between modules show how concepts connect
