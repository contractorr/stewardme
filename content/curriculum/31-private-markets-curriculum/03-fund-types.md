## 3.2 Private Equity: Buyout

### What It Is

**Buyout funds** acquire controlling stakes (often 100%) in established companies, improve them, and sell for a profit.

### Characteristics

| Attribute | Typical Range |
|-----------|--------------|
| **Ownership** | 51-100% (control) |
| **Company Stage** | Mature, profitable |
| **EBITDA** | $10M - $500M+ |
| **Hold Period** | 4-7 years |
| **Leverage** | 4-6x EBITDA |
| **Target Returns** | 15-25% net IRR, 2.0-2.5x MOIC |

### Value Creation Levers

```
┌─────────────────────────────────────────────────────────────┐
│              BUYOUT VALUE CREATION                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. OPERATIONAL IMPROVEMENT                                 │
│     • Revenue growth (new products, markets, pricing)       │
│     • Cost reduction (efficiency, procurement)              │
│     • Management upgrades                                   │
│     • Digital transformation                                │
│                                                             │
│  2. FINANCIAL ENGINEERING                                   │
│     • Leverage (debt magnifies equity returns)              │
│     • Working capital optimization                          │
│     • Tax efficiency                                        │
│                                                             │
│  3. MULTIPLE EXPANSION                                      │
│     • Buy at 8x EBITDA, sell at 10x EBITDA                 │
│     • Improve company quality/growth profile                │
│     • Better market conditions at exit                      │
│                                                             │
│  4. STRATEGIC TRANSACTIONS                                  │
│     • Add-on acquisitions                                   │
│     • Carve-outs and divestitures                          │
│     • Platform building                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Buyout Segments

| Segment | Company EBITDA | Fund Size | Example GPs |
|---------|---------------|-----------|-------------|
| **Mega/Large** | $200M+ | $10B+ | Blackstone, KKR, Apollo |
| **Upper Mid-Market** | $75-200M | $2-10B | Advent, Hellman & Friedman |
| **Mid-Market** | $25-75M | $500M-2B | GTCR, Summit Partners |
| **Lower Mid-Market** | $10-25M | $100M-500M | Many regional firms |
| **Small** | <$10M | <$100M | Smaller/local firms |

## 3.3 Private Equity: Growth Equity

### What It Is

**Growth equity** invests in established, profitable companies seeking capital for expansion - without the leverage of buyouts.

### How It Differs from Buyout

| Attribute | Growth Equity | Buyout |
|-----------|--------------|--------|
| **Ownership** | Minority (20-40%) | Control (51-100%) |
| **Company Profile** | Profitable, fast-growing | Mature, stable |
| **Leverage** | Little to none | Significant (4-6x) |
| **Use of Proceeds** | Growth (expand, hire, acquire) | Buyout of existing owners |
| **Risk Profile** | Lower (established businesses) | Leverage adds risk |

### Characteristics

| Attribute | Typical Range |
|-----------|--------------|
| **Ownership** | 15-40% minority |
| **Company Stage** | Profitable, growing 15-30%+ |
| **Revenue** | $20M - $200M+ |
| **Hold Period** | 4-6 years |
| **Leverage** | 0-1x |
| **Target Returns** | 20-30% IRR, 2.5-4.0x MOIC |

### Example Growth Equity Investment

```
TECHSOLUTIONS INC. - GROWTH EQUITY EXAMPLE
──────────────────────────────────────────

INVESTMENT THESIS
• B2B SaaS company with $50M ARR
• Growing 40% annually, profitable
• Needs capital to expand sales team and enter Europe
• Founders want to stay, not sell control

TRANSACTION
• $100M investment for 25% ownership
• Pre-money valuation: $300M
• Post-money valuation: $400M

EXIT (Year 5)
• Revenue grown to $300M ARR
• Exit at 8x revenue = $2.4B valuation
• 25% stake = $600M
• Return: 6.0x, ~43% IRR
```

## 3.4 Venture Capital

### What It Is

**Venture capital (VC)** provides funding to early-stage startups with high growth potential in exchange for equity.

### Key Characteristics

| Attribute | Typical Range |
|-----------|--------------|
| **Ownership** | 10-30% per round |
| **Company Stage** | Pre-revenue to early growth |
| **Revenue** | $0 - $50M |
| **Hold Period** | 5-10 years |
| **Leverage** | None |
| **Target Returns** | 25-35% IRR, 3x+ MOIC (fund level) |

### The VC Model: Power Law Returns

```
VC FUND RETURNS DISTRIBUTION (100 investments)
──────────────────────────────────────────────

  50% │ ████████████████████████████████████████████████
     │ │                  50 investments
     │ │                  Return: 0-1x (losses or breakeven)
     │ │
  30% │ ████████████████████████████████
     │ │              30 investments
     │ │              Return: 1-3x (modest returns)
     │ │
  15% │ ████████████████
     │ │        15 investments
     │ │        Return: 3-10x (good returns)
     │ │
   5% │ ████
     │ │    5 investments
     │ │    Return: 10-100x+ ("home runs")
     │ │    ▲
     │ │    └── These 5% drive most of fund returns
     └─┴────────────────────────────────────────────────►
              Return Multiple
```

**Key insight**: VC funds expect most investments to fail. A few massive winners drive overall returns.

### VC Stages

| Stage | Company State | Check Size | Valuation | Example Investors |
|-------|--------------|------------|-----------|-------------------|
| **Pre-Seed** | Idea/prototype | $100K-1M | $2-10M | Angels, micro VCs |
| **Seed** | MVP, early users | $1-5M | $5-20M | Seed funds |
| **Series A** | Product-market fit | $5-20M | $20-80M | Early-stage VCs |
| **Series B** | Scaling | $15-50M | $80-200M | Multi-stage VCs |
| **Series C+** | Expansion | $50-200M+ | $200M-$5B+ | Growth/crossover |

### VC Fund Terms

| Term | Typical | Notes |
|------|---------|-------|
| Management Fee | 2.0-2.5% | Higher than buyout |
| Carry | 20-30% | Top funds command 25-30% |
| Fund Life | 10-12 years | Longer than buyout |
| Investment Period | 3-4 years | Shorter, concentrated |

## 3.5 Real Assets: Real Estate

### What It Is

**Real estate private equity** invests in properties or property-related companies.

### Strategy Spectrum

```
RISK/RETURN SPECTRUM
────────────────────

                     Risk
         Low ◄─────────────────────► High
          │                            │
          │                            │
    CORE  │  CORE-PLUS  │  VALUE-ADD  │  OPPORTUNISTIC
    ──────┴─────────────┴─────────────┴──────────────

    • Stabilized        • Light         • Renovation     • Development
    • Trophy assets       renovation    • Repositioning  • Distressed
    • Long leases       • Some vacancy  • Lease-up       • Land
    • Low leverage      • Moderate      • Higher         • High leverage
      (30-50%)           leverage        leverage
                        (40-60%)        (60-75%)

    Target IRR:         Target IRR:     Target IRR:      Target IRR:
    6-9%               9-12%           12-18%           18%+
```

### Property Types

| Type | Description | Key Metrics |
|------|-------------|-------------|
| **Office** | Commercial office buildings | $/SF rent, occupancy |
| **Retail** | Shopping centers, malls | Sales/SF, occupancy |
| **Industrial** | Warehouses, logistics | $/SF, clear height |
| **Multifamily** | Apartment buildings | Rent/unit, occupancy |
| **Hospitality** | Hotels | RevPAR, ADR |
| **Data Centers** | IT infrastructure | $/kW, PUE |

### Real Estate-Specific Metrics

| Metric | Formula | Meaning |
|--------|---------|---------|
| **NOI** | Revenue - Operating Expenses | Property cash flow |
| **Cap Rate** | NOI / Property Value | Yield on property |
| **Occupancy** | Leased SF / Total SF | Utilization |
| **WALT** | Weighted Avg Lease Term | Income stability |

### Example Real Estate Investment

```
DOWNTOWN OFFICE TOWER - VALUE-ADD EXAMPLE
─────────────────────────────────────────

ENTRY
• Purchase Price: $100M
• NOI: $5M (5% cap rate)
• Occupancy: 70%
• Issues: Outdated lobby, poor amenities

BUSINESS PLAN
• Invest $15M in renovations
• Improve amenities (gym, conference)
• Modernize building systems
• Lease vacant space at higher rents

EXIT (Year 5)
• NOI: $8.5M (70% increase)
• Occupancy: 95%
• Exit Cap Rate: 5.5%
• Value: $8.5M / 5.5% = $155M

RETURNS
Total Invested: $115M (purchase + capex)
Exit Value: $155M
Profit: $40M
MOIC: 1.35x on total investment
Equity IRR: ~20% (with 65% leverage)
```

## 3.7 Private Credit

### What It Is

**Private credit** provides debt financing directly to companies, bypassing traditional bank lending.

### Why It Exists

After 2008 financial crisis:
- Banks reduced lending (regulatory capital requirements)
- Middle-market companies still need capital
- Private credit funds filled the gap
- Investors wanted yield in low-rate environment

### Strategy Types

```
┌─────────────────────────────────────────────────────────────┐
│                 PRIVATE CREDIT SPECTRUM                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  SENIOR SECURED ────► MEZZANINE ────► DISTRESSED           │
│  (Lowest Risk)                        (Highest Risk)        │
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ • First lien │  │ • Subordinate│  │ • Bankrupt/ │         │
│  │ • Asset-     │  │ • 2nd lien   │  │   stressed  │         │
│  │   backed     │  │ • Unsecured  │  │   companies │         │
│  │             │  │ • PIK        │  │ • Turnaround│         │
│  │ Yield: 8-12%│  │ Yield: 12-18%│  │ Returns:    │         │
│  │             │  │             │  │  20%+ IRR   │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Key Characteristics

| Attribute | Private Credit |
|-----------|---------------|
| **Return Profile** | Current income + modest capital appreciation |
| **Typical Yield** | 8-15% |
| **Loan Terms** | 3-7 years |
| **Covenants** | Tighter than public debt |
| **Documentation** | More lender-friendly |

### Direct Lending vs Mezzanine

| Aspect | Direct Lending | Mezzanine |
|--------|---------------|-----------|
| **Priority** | Senior (first to be paid) | Junior (after senior) |
| **Security** | Secured by assets | Often unsecured |
| **Yield** | 8-12% | 12-18%+ |
| **Upside** | Limited | Warrants/equity kickers |

## 3.8 Fund of Funds

### What It Is

A **Fund of Funds (FoF)** is a fund that invests in other private equity funds rather than directly in companies.

### Structure

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                    FUND OF FUNDS                            │
│                   ($500M total)                             │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                   INVESTOR (LP)                      │  │
│   │            Commits $10M to FoF                       │  │
│   └──────────────────────┬──────────────────────────────┘  │
│                          │                                  │
│                          ▼                                  │
│   ┌─────────────────────────────────────────────────────┐  │
│   │              FUND OF FUNDS MANAGER                   │  │
│   │         Allocates across 20-30 funds                 │  │
│   └───────────┬────────┬────────┬────────┬──────────────┘  │
│               │        │        │        │                  │
│        ┌──────┴─┐  ┌───┴───┐  ┌─┴────┐  ┌┴──────┐          │
│        ▼        ▼  ▼       ▼  ▼      ▼  ▼       ▼          │
│    ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐      │
│    │Buyout │ │Buyout │ │  VC   │ │  VC   │ │Growth │      │
│    │Fund A │ │Fund B │ │Fund C │ │Fund D │ │Fund E │      │
│    │ $30M  │ │ $25M  │ │ $20M  │ │ $15M  │ │ $20M  │      │
│    └───────┘ └───────┘ └───────┘ └───────┘ └───────┘      │
│                                                             │
│    ... and 15-25 more underlying funds ...                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Why Invest Through FoF?

| Benefit | Explanation |
|---------|-------------|
| **Diversification** | Exposure to 20-30+ funds, hundreds of companies |
| **Access** | Top-tier funds often closed to new investors |
| **Lower Minimums** | $1-5M vs $10-25M direct |
| **Professional Selection** | Expert due diligence on fund managers |
| **Simplified Admin** | One relationship vs. 20+ |

### The Double Fee Layer

```
FoF FEE STRUCTURE
─────────────────

UNDERLYING FUNDS (20-30 funds)
• Management Fee: 1.5-2.0%
• Carry: 20%

        +

FUND OF FUNDS LAYER
• Management Fee: 0.5-1.0%
• Carry: 5-10%

        =

TOTAL FEES
• ~2.0-3.0% management fee equivalent
• ~25-30% effective carry

IMPACT ON RETURNS
• Underlying funds return 15% gross
• After underlying fees: ~12% net
• After FoF fees: ~10% net to investor
```

### FoF vs Direct Investing

| Aspect | Fund of Funds | Direct LP |
|--------|--------------|-----------|
| **Minimum** | $1-5M | $10-25M |
| **Diversification** | Immediate (20+ funds) | Build over time |
| **Access** | Leverage FoF relationships | Must earn access |
| **Fees** | Double layer | Single layer |
| **Control** | None | Some (via LPAC, etc.) |
| **Returns** | Lower (after fees) | Higher potential |

## 3.9 Secondary Funds

### What It Is

**Secondary funds** buy existing LP interests in private equity funds from investors who want to exit early.

### How It Works

```
SECONDARY TRANSACTION
─────────────────────

ORIGINAL SITUATION
┌─────────────┐         ┌─────────────┐
│  Original   │ ──────► │    Fund     │
│     LP      │ $50M    │             │
│  (Seller)   │ commit  │   Year 4    │
└─────────────┘         │   NAV: $55M │
                        └─────────────┘

SECONDARY SALE
┌─────────────┐         ┌─────────────┐
│  Secondary  │ ──────► │  Original   │
│    Fund     │  $49M   │     LP      │
│   (Buyer)   │  (89%   │  (Seller)   │
│             │ of NAV) │             │
└─────────────┘         └─────────────┘
      │                         │
      │    Secondary fund       │ Original LP
      │    now owns the         │ gets liquidity
      │    $50M commitment      │ (at discount)
      │    and $55M NAV         │
      ▼                         │
┌─────────────┐                 │
│    Fund     │ ◄───────────────┘
│             │   Transfer
│   Year 4    │   of LP interest
└─────────────┘
```

### Why Secondaries Exist

**Sellers want**:
- Liquidity before fund ends
- Portfolio rebalancing
- Regulatory/strategic changes
- Cash needs

**Buyers get**:
- Mature assets (reduced J-curve)
- Discounts to NAV
- Immediate distributions
- Known portfolio (vs. blind pool)

### Typical Pricing

| Market Condition | Discount to NAV |
|-----------------|-----------------|
| **Strong Market** | 0-5% |
| **Normal Market** | 5-15% |
| **Stressed Market** | 15-30%+ |

## 3.11 Identifying Fund Type from Data

When looking at LP reports, you can often identify fund type by:

### Key Indicators

| Data Point | What It Suggests |
|------------|-----------------|
| **High leverage ratios** | Buyout |
| **EV/EBITDA multiples** | Buyout or Growth |
| **Many small investments** | Venture Capital |
| **Occupancy/WALT** | Real Estate |
| **Yield/coupon data** | Private Credit |
| **Interest income primary** | Credit fund |
| **Multiple underlying funds** | Fund of Funds |
| **NOI, Cap Rate** | Real Estate |

## Knowledge Check

1. What's the main difference between buyout and growth equity?
2. Why do VC funds expect most investments to fail?
3. What's the difference between "core" and "value-add" real estate?
4. Why would an LP invest in a fund of funds despite higher fees?
5. What does a secondary fund buy?

<details>
<summary>Answers</summary>

1. Buyout acquires control (51-100%) using leverage; growth equity takes minority stakes (15-40%) with little/no leverage
2. VC follows a power law - a few massive winners (10-100x) compensate for many failures. This is expected and modeled into returns.
3. Core = stabilized, low-risk, cash-flowing assets. Value-add = assets needing improvement (renovation, lease-up) with higher risk/return.
4. Diversification, access to top funds, lower minimums, professional selection, simplified administration
5. Existing LP interests in funds from investors wanting liquidity before fund ends

</details>
