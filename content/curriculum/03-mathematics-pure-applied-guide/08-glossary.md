# Comprehensive Mathematics Glossary

## A

**Abelian Group**: Group where operation is commutative (a • b = b • a for all elements)

**Absolute Value**: Distance from zero; |x| = x if x ≥ 0, |x| = -x if x < 0

**Abstract Algebra**: Study of algebraic structures (groups, rings, fields) independent of specific representations

**Accumulation**: Total quantity obtained by summing or integrating; fundamental concept underlying integration

**Addition Principle**: If choosing from disjoint options, total possibilities = sum of individual possibilities

**Algebraic Number**: Root of polynomial equation with integer coefficients (includes rationals, √2, i)

**Algorithm**: Step-by-step procedure for solving problem or performing computation

**Analysis**: Branch studying limits, continuity, derivatives, integrals with mathematical rigor

**Analytic Geometry**: Using coordinates and equations to study geometric objects (Cartesian plane)

**Angle**: Measure of rotation between two rays sharing common endpoint (measured in degrees or radians)

**Antiderivative**: Function F(x) whose derivative is f(x); also called indefinite integral

**Associative**: Property where (a • b) • c = a • (b • c); order of operations doesn't matter

**Asymptote**: Line that curve approaches but never reaches as x → ±∞

**Axiom**: Statement accepted as true without proof; foundation of mathematical theory

**Axiomatic System**: Mathematical structure built from axioms using logical deduction

## B

**Basis**: Linearly independent set of vectors spanning entire vector space

**Bayes' Theorem**: P(A|B) = P(B|A)P(A)/P(B); updates probabilities given new evidence

**Bernoulli Trial**: Experiment with exactly two outcomes (success/failure)

**Biconditional**: Statement "P if and only if Q" (P ⟺ Q); true when both have same truth value

**Bifurcation**: Qualitative change in dynamical system's behavior as parameter varies

**Big O Notation**: Describes asymptotic behavior of functions; characterizes algorithm efficiency

**Bijection**: Function that is both one-to-one (injective) and onto (surjective); establishes one-to-one correspondence

**Binary Operation**: Operation combining two elements to produce third (addition, multiplication)

**Binomial Coefficient**: C(n,k) = n!/(k!(n-k)!); "n choose k"; counts combinations

**Binomial Theorem**: (x + y)ⁿ = Σₖ₌₀ⁿ C(n,k)xⁿ⁻ᵏyᵏ; expands polynomial powers

**Bipartite Graph**: Graph whose vertices can be divided into two sets with edges only between sets

**Boundary Condition**: Constraint specifying function values at boundaries; needed to solve differential equations

**Bounded**: Set/function with finite upper and lower limits

**Brownian Motion**: Continuous random walk; mathematical model of random particle movement

## C

**Calculus**: Mathematics of change and accumulation; studies derivatives and integrals

**Cantor's Diagonal Argument**: Proof that real numbers are uncountable (larger infinity than integers)

**Cardinality**: Size of set; number of elements (can be finite or infinite)

**Cartesian Product**: A × B = {(a,b) : a ∈ A, b ∈ B}; set of all ordered pairs

**Category Theory**: Very abstract branch unifying mathematics through study of objects and morphisms

**Cauchy Sequence**: Sequence where terms become arbitrarily close to each other; used to define convergence rigorously

**Central Limit Theorem**: Sum of many independent random variables approaches normal distribution

**Chain Rule**: For composite function: (f ∘ g)'(x) = f'(g(x)) · g'(x); enables differentiation of compositions

**Chaos Theory**: Study of systems exhibiting sensitive dependence on initial conditions

**Characteristic Equation**: Equation det(A - λI) = 0 used to find eigenvalues

**Chromatic Number**: Minimum colors needed to color graph vertices so adjacent vertices differ

**Circle**: Set of points equidistant from center; equation (x-h)² + (y-k)² = r²

**Closed Set**: Set containing all its boundary points (complement of open set)

**Coefficient**: Numerical factor multiplying variable or term in expression

**Cofactor**: Signed minor of matrix; used in computing determinants

**Combination**: Unordered selection; C(n,k) ways to choose k from n items

**Commutative**: Property where a • b = b • a; order doesn't matter

**Compact**: Set that is closed and bounded (in metric spaces); generalizes "finite" to infinite sets

**Complement**: Elements not in set; Aᶜ = {x : x ∉ A}

**Complex Number**: Number of form a + bi where i² = -1; elements of ℂ

**Composite Function**: (f ∘ g)(x) = f(g(x)); applying one function then another

**Concave**: Curve bending downward (f''(x) < 0); segment between points lies below curve

**Conditional Probability**: P(A|B); probability of A given B has occurred

**Confidence Interval**: Range likely to contain true parameter value with specified confidence level

**Conic Section**: Curve formed by intersecting plane with cone (circle, ellipse, parabola, hyperbola)

**Conjecture**: Statement believed true but not yet proven

**Conjugate**: For complex number a+bi, conjugate is a-bi; for expression with radical, rationalize by multiplying

**Connected**: Graph/space where path exists between any two points

**Constant**: Value that doesn't change; function f(x) = c has derivative 0

**Constraint**: Restriction or limitation in optimization problem

**Continuity**: Function is continuous at a if lim[x→a] f(x) = f(a); no jumps or breaks

**Continuum**: Range of values forming continuous whole (like real number line)

**Continuum Hypothesis**: No set has cardinality strictly between integers and reals; proven independent of ZFC axioms

**Contrapositive**: Statement ¬Q ⇒ ¬P equivalent to P ⇒ Q; useful proof technique

**Convergence**: Sequence/series approaches finite limit as n → ∞

**Convex**: Curve bending upward (f''(x) > 0); segment between points lies above curve

**Coordinate**: Number specifying position along axis; ordered n-tuple specifies point in n-dimensional space

**Corollary**: Theorem following easily from previous result

**Correlation**: Statistical measure of relationship between variables (-1 to 1)

**Cosine**: Trigonometric function; cos(θ) = adjacent/hypotenuse in right triangle

**Countable**: Set having same cardinality as natural numbers (can be listed)

**Counterexample**: Specific instance showing statement is false

**Cross Product**: Vector operation in 3D; **u** × **v** perpendicular to both, magnitude ||**u**||||**v**||sin(θ)

**Cryptography**: Study of secure communication; heavily uses number theory and algebra

**Curl**: Vector operator measuring rotation; ∇ × **F** for vector field **F**

**Curvature**: Measure of how quickly curve bends; κ = ||d**T**/ds|| where **T** is unit tangent

**Cycle**: Closed path in graph returning to starting vertex

## D

**Definite Integral**: ∫ₐᵇ f(x)dx; signed area under curve from a to b

**Degree**: (1) Angle measure (360° in full circle) (2) Highest power in polynomial (3) Number of edges incident to graph vertex

**Degree of Freedom**: Number of independent parameters or variables

**Derivative**: Instantaneous rate of change; f'(x) = lim[h→0] [f(x+h) - f(x)]/h

**Determinant**: Scalar value encoding properties of square matrix; det(A); relates to volume scaling and invertibility

**Diagonal Matrix**: Square matrix with non-zero entries only on main diagonal

**Diameter**: (1) Longest distance across circle (2d) (2) Maximum distance between any two vertices in graph

**Difference Equation**: Discrete analog of differential equation; relates values at different time steps

**Differentiable**: Function having derivative at point; implies continuity but not vice versa

**Differential**: Small change in quantity; dy = f'(x)dx

**Differential Equation**: Equation involving function and its derivatives; ODE or PDE

**Differential Geometry**: Study of curves and surfaces using calculus

**Dimension**: Number of coordinates needed to specify point; number of vectors in basis

**Directed Graph (Digraph)**: Graph where edges have direction (arrows)

**Dirichlet Problem**: Find function satisfying PDE in region with specified boundary values

**Discrete**: Consisting of distinct, separated values (opposite of continuous)

**Discrete Mathematics**: Math of countable structures (logic, combinatorics, graphs)

**Discriminant**: Δ = b² - 4ac in quadratic formula; determines nature of roots

**Distributive**: Property a(b + c) = ab + ac; multiplication distributes over addition

**Divergence**: (1) Sequence/series not converging to finite limit (2) Vector operator ∇ · **F** measuring outflow

**Divisibility**: a divides b (a|b) if b = ka for some integer k

**Domain**: Set of input values for function; f: A → B has domain A

**Dot Product**: **u** · **v** = u₁v₁ + ... + uₙvₙ; measures similarity, projects vectors

**Dual Space**: Space of linear functionals on vector space

**Dynamical System**: Mathematical model describing time evolution of system in state space

## E

**e (Euler's Number)**: Mathematical constant ≈ 2.71828; base of natural logarithm; lim[n→∞](1 + 1/n)ⁿ

**Eigenvalue**: Scalar λ such that A**v** = λ**v** for some non-zero vector **v**

**Eigenvector**: Non-zero vector **v** scaled (not rotated) by linear transformation: A**v** = λ**v**

**Element**: Member of set; 3 ∈ {1,2,3}

**Elementary Function**: Built from polynomials, exponentials, logarithms, trig functions using arithmetic and composition

**Ellipse**: Conic section; sum of distances to two foci is constant; (x²/a²) + (y²/b²) = 1

**Elliptic Geometry**: Non-Euclidean geometry with no parallel lines (spherical geometry)

**Empty Set**: Set with no elements; ∅ or {}

**Equation**: Mathematical statement asserting equality of two expressions

**Equilibrium**: State where system doesn't change; dx/dt = 0

**Equivalence Relation**: Relation that is reflexive, symmetric, and transitive (partitions set into classes)

**Error Function**: erf(x) = (2/√π)∫₀ˣ e⁻ᵗ² dt; arises in probability and PDEs

**Euclidean Algorithm**: Efficient method to compute GCD using repeated division

**Euclidean Geometry**: Geometry based on Euclid's axioms; flat space

**Euclidean Space**: ℝⁿ with standard distance metric

**Euler Characteristic**: χ = V - E + F for polyhedron; topological invariant

**Euler's Formula**: e^(iθ) = cos(θ) + i sin(θ); connects exponential and trigonometric functions

**Euler's Identity**: e^(iπ) + 1 = 0; called "most beautiful equation"

**Euler Path/Circuit**: Path/circuit in graph using each edge exactly once

**Even Function**: f(-x) = f(x); symmetric about y-axis

**Event**: Subset of sample space in probability

**Expected Value**: Average outcome weighted by probabilities; E[X] = Σ x P(X=x)

**Exponent**: Power to which number is raised; in xⁿ, n is exponent

**Exponential Function**: f(x) = aˣ (especially e^x); grows/decays at rate proportional to current value

**Extremum**: Maximum or minimum value (local or global)

## F

**Factorial**: n! = n × (n-1) × ... × 2 × 1; number of permutations of n objects

**Factoring**: Writing expression as product (e.g., x² - 4 = (x-2)(x+2))

**Feasible Region**: Set of points satisfying all constraints in optimization

**Fermat's Last Theorem**: xⁿ + yⁿ = zⁿ has no integer solutions for n > 2; proved by Wiles (1995)

**Fermat's Little Theorem**: If p prime and a not divisible by p, then a^(p-1) ≡ 1 (mod p)

**Fibonacci Sequence**: 1, 1, 2, 3, 5, 8, 13, ...; each term is sum of previous two

**Field**: Ring where every non-zero element has multiplicative inverse (e.g., ℚ, ℝ, ℂ)

**Fixed Point**: Point unchanged by function; f(x) = x

**Forcing Function**: External input or driving term in differential equation

**Fourier Analysis**: Decomposing functions into sums of sines and cosines

**Fourier Series**: Representation of periodic function as sum of sinusoids

**Fourier Transform**: Extension of Fourier series to non-periodic functions; maps time to frequency domain

**Fractal**: Self-similar structure at multiple scales; fractional dimension

**Frequency**: Number of cycles per unit time; reciprocal of period

**Function**: Rule assigning each input exactly one output; f: A → B

**Functional**: Function whose inputs are functions (e.g., integrals)

**Fundamental Theorem of Algebra**: Every polynomial of degree n has exactly n complex roots (counting multiplicity)

**Fundamental Theorem of Arithmetic**: Every integer > 1 has unique prime factorization

**Fundamental Theorem of Calculus**: Connects differentiation and integration; ∫ₐᵇ f'(x)dx = f(b) - f(a)

## G

**Galois Theory**: Studies symmetries of polynomial roots; shows no formula for general quintic

**Gauss-Jordan Elimination**: Method to solve systems and find matrix inverses using row operations

**Gaussian Elimination**: Algorithm to solve systems of linear equations by row reduction

**GCD (Greatest Common Divisor)**: Largest integer dividing both a and b

**General Relativity**: Einstein's theory where gravity is spacetime curvature; uses Riemannian geometry

**Genus**: Number of "holes" in topological surface; torus has genus 1

**Geodesic**: Shortest path in curved space; generalizes straight line

**Geometric Mean**: nth root of product of n numbers; √(x₁x₂...xₙ)

**Geometric Series**: Σ arⁿ; converges to a/(1-r) if |r| < 1

**Geometry**: Study of shapes, sizes, positions, and spatial properties

**Gödel's Incompleteness Theorems**: Any consistent formal system containing arithmetic has unprovable true statements

**Gradient**: Vector of partial derivatives; ∇f = (∂f/∂x, ∂f/∂y, ...); points in direction of steepest increase

**Gradient Descent**: Iterative optimization algorithm following negative gradient

**Graph**: (1) Visual representation of function (2) Structure G = (V,E) of vertices and edges

**Graph Theory**: Study of graphs (networks); vertices connected by edges

**Greatest Lower Bound (Infimum)**: Largest value that is ≤ all elements in set

**Group**: Set with associative operation, identity element, and inverses for all elements

**Group Theory**: Study of symmetry through algebraic structures

## H

**Hamilton Path/Circuit**: Path/circuit visiting each vertex exactly once

**Harmonic Series**: 1 + 1/2 + 1/3 + 1/4 + ...; diverges despite terms → 0

**Harmonic Function**: Solution to Laplace's equation; Δu = 0

**Heat Equation**: PDE modeling temperature diffusion; ∂u/∂t = α∇²u

**Hessian Matrix**: Matrix of second partial derivatives; used in optimization

**Heuristic**: Problem-solving approach using practical methods (not guaranteed optimal)

**Hilbert Space**: Complete inner product space; generalization of Euclidean space (quantum mechanics)

**Homeomorphism**: Continuous bijection with continuous inverse; establishes topological equivalence

**Homogeneous**: (1) Differential equation with zero right side (2) Polynomial where all terms have same degree

**Homology**: Algebraic topology tool detecting holes of various dimensions

**Hyperbola**: Conic section; difference of distances to two foci is constant; (x²/a²) - (y²/b²) = 1

**Hyperbolic Geometry**: Non-Euclidean geometry with infinitely many parallels through external point

**Hypothesis**: Tentative explanation or assumption to be tested

**Hypothesis Testing**: Statistical method to test claims about populations using sample data

## I

**i (Imaginary Unit)**: Defined by i² = -1; basis for complex numbers

**Identity**: Element e where a • e = e • a = a for all a (0 for addition, 1 for multiplication)

**Identity Matrix**: Square matrix with 1s on diagonal, 0s elsewhere; AI = IA = A

**Iff (If and Only If)**: Biconditional; P ⟺ Q means P ⇒ Q and Q ⇒ P

**Image**: Set of output values of function; f(A) = {f(x) : x ∈ A}

**Imaginary Number**: Complex number with zero real part; purely bi for b ≠ 0

**Implication**: Logical connective P ⇒ Q; "if P then Q"; false only when P true and Q false

**Implicit Differentiation**: Finding derivative when y not explicitly given as function of x

**Improper Integral**: Integral with infinite limits or discontinuous integrand

**Incidence Matrix**: Matrix representing graph structure; rows = vertices, columns = edges

**Inclusion-Exclusion Principle**: |A ∪ B| = |A| + |B| - |A ∩ B|; generalizes to more sets

**Indefinite Integral**: Antiderivative; ∫f(x)dx = F(x) + C

**Independence**: (1) Events: P(A∩B) = P(A)P(B) (2) Vectors: c₁**v**₁ + ... + cₖ**vₖ** = **0** only if all cᵢ = 0

**Index**: (1) Subscript identifying element (2) Variable in summation

**Induction**: Proof technique: show base case, then prove n ⇒ n+1

**Inequality**: Statement comparing expressions using <, ≤, >, or ≥

**Infimum**: Greatest lower bound of set

**Infinite Series**: Sum of infinitely many terms; Σ aₙ

**Infinity**: Concept of unboundedness; symbol ∞

**Inflection Point**: Where concavity changes; f''(x) = 0 and changes sign

**Injection (One-to-One)**: Function where different inputs give different outputs

**Inner Product**: Generalization of dot product; defines angles and lengths in vector space

**Integer**: Whole number (positive, negative, or zero); ℤ = {..., -2, -1, 0, 1, 2, ...}

**Integrable**: Function whose integral exists

**Integral**: (1) Area under curve (definite) (2) Antiderivative (indefinite)

**Integration by Parts**: ∫u dv = uv - ∫v du; based on product rule

**Integration by Substitution**: Change of variables in integral; u-substitution

**Interpolation**: Estimating values between known data points

**Intersection**: A ∩ B = {x : x ∈ A and x ∈ B}; elements in both sets

**Interval**: Set of real numbers between two endpoints; [a,b], (a,b), etc.

**Invariant**: Property unchanged under transformation

**Inverse**: (1) Element a⁻¹ where a • a⁻¹ = e (2) Inverse function f⁻¹ where f(f⁻¹(x)) = x

**Inverse Matrix**: A⁻¹ such that AA⁻¹ = A⁻¹A = I

**Invertible**: Having an inverse

**Irrational Number**: Real number not expressible as ratio of integers (π, e, √2)

**Isomorphism**: Structure-preserving bijection between mathematical objects

**Iteration**: Repeated application of procedure; xₙ₊₁ = f(xₙ)

**Itô Calculus**: Calculus for stochastic processes; foundation for mathematical finance

## J

**Jacobian**: Matrix of first partial derivatives; determinant measures volume scaling in change of variables

**Jordan Curve**: Simple closed curve (no self-intersections)

**Julia Set**: Fractal associated with complex function; boundary of stable region

## K

**Kernel**: (1) Null space of linear transformation (2) Function in integral transform

**Klein Bottle**: Non-orientable surface; generalization of Möbius strip

**Knot**: Closed loop embedded in 3D space

**Knot Theory**: Study of mathematical knots and their classification

**Kronecker Delta**: δᵢⱼ = 1 if i = j, 0 otherwise

## L

**Lagrange Multiplier**: Method for constrained optimization; λ such that ∇f = λ∇g at optimum

**Laplace Transform**: Integral transform; maps function of t to function of s; solves ODEs

**Laplacian**: Operator ∇²f = ∂²f/∂x² + ∂²f/∂y² + ...; measures local deviation from average

**Lattice**: (1) Regular grid in space (2) Partially ordered set with meets and joins

**Law of Large Numbers**: Sample average converges to expected value as sample size → ∞

**LCM (Least Common Multiple)**: Smallest positive integer divisible by both a and b

**Least Squares**: Method minimizing sum of squared residuals; **w** = (XᵀX)⁻¹Xᵀ**y**

**Lemma**: Minor theorem, typically used to prove larger result

**L'Hôpital's Rule**: For indeterminate forms 0/0 or ∞/∞: lim f/g = lim f'/g'

**Limit**: Value function approaches; lim[x→a] f(x) = L

**Linear**: (1) Degree 1 (straight line) (2) Preserving addition and scalar multiplication

**Linear Algebra**: Study of vectors, matrices, linear transformations

**Linear Combination**: Expression c₁**v**₁ + c₂**v**₂ + ... + cₙ**vₙ**

**Linear Independence**: Vectors where no one is combination of others

**Linear Programming**: Optimization of linear objective with linear constraints

**Linear Transformation**: Function T where T(**u** + **v**) = T(**u**) + T(**v**) and T(c**v**) = cT(**v**)

**Logarithm**: Inverse of exponential; log_a(x) = y ⟺ aʸ = x

**Logic**: Study of valid reasoning and proof

**Logistic Function**: S-shaped curve; f(x) = L/(1 + e⁻ᵏ⁽ˣ⁻ˣ⁰⁾); models growth with saturation

**Loop**: Edge connecting vertex to itself in graph

## M

**Manifold**: Space locally resembling Euclidean space; generalization of curves and surfaces

**Mapping**: Another term for function

**Markov Chain**: Stochastic process where future depends only on present, not past

**Mathematical Induction**: Proof technique: base case + (P(n) ⇒ P(n+1))

**Matrix**: Rectangular array of numbers; represents linear transformation

**Maximum**: Largest value in set or of function

**Mean**: Average; sum divided by count; μ = (x₁ + ... + xₙ)/n

**Mean Value Theorem**: If f continuous on [a,b] and differentiable on (a,b), then ∃c where f'(c) = [f(b)-f(a)]/(b-a)

**Measure**: Generalization of length, area, volume; foundation for integration theory

**Median**: Middle value when data ordered; 50th percentile

**Metric**: Distance function satisfying d(x,y) ≥ 0, d(x,x) = 0, symmetry, triangle inequality

**Metric Space**: Set with metric defining distances

**Minimum**: Smallest value in set or of function

**Minor**: Determinant of submatrix formed by deleting row and column

**Möbius Strip**: Non-orientable surface with one side and one edge

**Mode**: Most frequently occurring value in data set

**Modular Arithmetic**: Arithmetic with remainders; a ≡ b (mod n) means n|(a-b)

**Modulus**: (1) Absolute value (2) In modular arithmetic: the n in "mod n" (3) For complex number: distance from origin

**Moment**: Expected value of power of random variable; E[Xⁿ]

**Monomial**: Single term expression; axⁿ

**Monotonic**: Always increasing or always decreasing; never changes direction

**Monte Carlo Method**: Using random sampling to solve problems numerically

**Morphism**: Structure-preserving map between mathematical objects

**Multiplication Principle**: Number of ways to perform sequence of tasks = product of ways for each task

**Multiplicity**: Number of times root appears; (x-2)³ has root 2 with multiplicity 3

**Multivariate**: Involving multiple variables

**Mutually Exclusive**: Events that cannot occur simultaneously; P(A∩B) = 0

## N

**Natural Number**: Positive integer; ℕ = {1, 2, 3, ...} (sometimes includes 0)

**Necessary Condition**: B is necessary for A if A ⇒ B (A cannot hold without B)

**Negation**: Logical NOT; ¬P has opposite truth value of P

**Neighborhood**: Set of points near given point; open ball in metric space

**Neural Network**: Computational model inspired by brain; composition of linear transformations and nonlinearities

**Newton's Method**: Root-finding algorithm; xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ)

**Noether's Theorem**: Continuous symmetries correspond to conservation laws (physics)

**Non-Euclidean Geometry**: Geometry where parallel postulate fails (hyperbolic or elliptic)

**Norm**: Generalization of length; ||**v**|| = √(**v** · **v**)

**Normal Distribution**: Bell-shaped probability distribution; N(μ, σ²)

**Normal Vector**: Vector perpendicular to surface or curve

**NP-Complete**: Hardest problems in NP; all NP problems reduce to these

**Null Hypothesis**: Default assumption in statistical testing (typically "no effect")

**Null Space**: Set of vectors mapped to zero by linear transformation; kernel

**Number Theory**: Study of integers and their properties

**Numerical Analysis**: Study of algorithms for continuous mathematics problems

**Numerical Method**: Algorithm approximating solutions to mathematical problems

## O

**Objective Function**: Function to optimize (maximize or minimize)

**Odd Function**: f(-x) = -f(x); symmetric about origin

**ODE (Ordinary Differential Equation)**: Equation involving function of one variable and its derivatives

**One-to-One**: Function where different inputs give different outputs; injective

**Onto**: Function where every element of codomain is output for some input; surjective

**Open Set**: Set not containing its boundary points

**Operator**: Symbol representing mathematical operation or transformation

**Optimization**: Finding best solution (maximum or minimum) from feasible set

**Order**: (1) Arrangement (2) Highest derivative in differential equation (3) Size of group

**Ordered Pair**: (a, b) where order matters; (1,2) ≠ (2,1)

**Ordinal**: Number indicating position (1st, 2nd, 3rd, ...)

**Origin**: Point (0, 0, ..., 0); reference point for coordinate system

**Orthogonal**: Perpendicular; **u** ⊥ **v** ⟺ **u** · **v** = 0

**Orthogonal Matrix**: Square matrix where QᵀQ = I; preserves lengths and angles

**Orthonormal**: Orthogonal unit vectors

**Oscillation**: Repeated back-and-forth variation

## P

**Parabola**: Conic section; graph of quadratic function; y = ax² + bx + c

**Paradox**: Statement leading to self-contradiction or counterintuitive result

**Parallel**: (1) Lines never intersecting (2) Vectors pointing same/opposite direction

**Parameter**: (1) Variable in parametric equation (2) Value characterizing system/distribution

**Parametric Equations**: Expressing coordinates as functions of parameter; x = f(t), y = g(t)

**Partial Derivative**: Derivative with respect to one variable, others held constant; ∂f/∂x

**Partial Differential Equation (PDE)**: Equation with partial derivatives; multiple independent variables

**Partition**: Division of set into non-overlapping subsets whose union is original set

**Pascal's Triangle**: Triangular array of binomial coefficients

**Path**: Sequence of edges connecting vertices in graph

**PDE**: See Partial Differential Equation

**Percentile**: Value below which given percentage of data falls

**Period**: (1) Interval after which function repeats (2) Decimal expansion cycle

**Periodic**: Repeating at regular intervals; f(x + p) = f(x) for all x

**Permutation**: Ordered arrangement; P(n,k) = n!/(n-k)!

**Perpendicular**: Meeting at right angle; orthogonal

**Phase Portrait**: Visualization of trajectories in state space for dynamical system

**Phase Space**: Space of all possible states of system

**π (Pi)**: Ratio of circle's circumference to diameter; ≈ 3.14159...

**Piecewise**: Function defined differently on different intervals

**Pigeonhole Principle**: If n+1 items placed in n boxes, at least one box contains ≥2 items

**Planar Graph**: Graph drawable on plane without edge crossings

**Plane**: Two-dimensional flat surface; equation ax + by + cz = d

**Poincaré Conjecture**: Every simply connected closed 3-manifold is homeomorphic to 3-sphere (now theorem; proved 2003)

**Point**: Location in space; no dimension

**Polar Coordinates**: (r, θ) representation; r = distance from origin, θ = angle from positive x-axis

**Polynomial**: Expression with terms of form aₙxⁿ; sum of monomials

**Population**: Entire group being studied in statistics

**Power Set**: Set of all subsets; 𝒫(A); if |A| = n, then |𝒫(A)| = 2ⁿ

**Power Series**: Σ aₙxⁿ; polynomial with infinitely many terms

**Predicate**: Statement depending on variable(s); becomes proposition when variables specified

**Prime Factorization**: Writing number as product of primes; unique by Fundamental Theorem of Arithmetic

**Prime Number**: Integer > 1 with only divisors 1 and itself

**Primitive**: Antiderivative; function whose derivative is given function

**Principal Component Analysis (PCA)**: Dimensionality reduction using eigenvectors of covariance matrix

**Probability**: Measure of likelihood; value between 0 and 1

**Product Rule**: (fg)' = f'g + fg'; for derivatives of products

**Projection**: Component of one vector along another; shadow cast

**Proof**: Logical argument establishing truth of statement from axioms and prior results

**Proof by Contradiction**: Assume negation, derive impossibility, conclude original statement true

**Proof by Induction**: Show base case and n ⇒ n+1

**Proper Subset**: A ⊂ B where A ⊆ B and A ≠ B

**Proposition**: Statement that is either true or false

**Pythagorean Theorem**: In right triangle: a² + b² = c² where c is hypotenuse

## Q

**Quadratic**: Degree 2; ax² + bx + c

**Quadratic Formula**: x = (-b ± √(b² - 4ac))/(2a); solves ax² + bx + c = 0

**Quantifier**: Symbol ∀ (for all) or ∃ (there exists)

**Quantum Mechanics**: Physics theory where observables are linear operators; eigenvalues = measurement outcomes

**Quartic**: Degree 4 polynomial

**Quaternion**: Extension of complex numbers to 4 dimensions; used in 3D rotations

**Quotient**: Result of division; a ÷ b

**Quotient Rule**: (f/g)' = (f'g - fg')/g²; for derivatives of quotients

## R

**Radian**: Angle measure; 2π radians = 360°; arc length = radius × angle in radians

**Radius**: Distance from center to edge of circle

**Random Variable**: Variable whose value is determined by random process

**Range**: (1) Set of output values of function (2) Difference between max and min in data

**Rank**: Dimension of column space; number of linearly independent rows/columns

**Rate of Change**: How quickly quantity changes; measured by derivative

**Ratio**: Comparison of two quantities by division; a/b

**Rational Function**: Ratio of polynomials; P(x)/Q(x)

**Rational Number**: Expressible as ratio of integers; ℚ = {p/q : p, q ∈ ℤ, q ≠ 0}

**Real Number**: Point on number line; ℝ includes rationals and irrationals

**Recursion**: Defining object in terms of itself; function calling itself

**Recursive Formula**: Formula expressing term using previous terms

**Reflexive**: Relation where a ~ a for all a

**Regression**: Statistical method modeling relationship between variables

**Relation**: Set of ordered pairs; generalizes function

**Relative**: Compared to something else; relative maximum only maximum in neighborhood

**Remainder**: What's left after division; a = bq + r where 0 ≤ r < b

**Residual**: Difference between observed and predicted value

**Riemann Hypothesis**: All non-trivial zeros of zeta function have real part 1/2; major unsolved problem

**Riemann Integral**: Definite integral as limit of Riemann sums (rectangles approximating area)

**Riemannian Geometry**: Geometry of curved spaces using metrics; foundation for general relativity

**Right Angle**: 90° angle; π/2 radians

**Ring**: Algebraic structure with addition and multiplication; integers form ring

**Root**: (1) Solution to equation f(x) = 0 (2) nth root: xⁿ = a

**Rotation**: Transformation turning object around fixed point/axis

**Rounding**: Approximating number to fewer digits

**Row Echelon Form**: Matrix form with leading 1s in staircase pattern

**RSA**: Public-key cryptography algorithm based on factoring difficulty

## S

**Saddle Point**: Critical point that is neither maximum nor minimum; f''(x) = 0, changes sign

**Sample**: Subset of population used for statistical inference

**Sample Space**: Set of all possible outcomes in probability experiment

**Scalar**: Single number (as opposed to vector or matrix)

**Scalar Multiplication**: Multiplying vector/matrix by number

**Scientific Notation**: Writing numbers as a × 10ⁿ where 1 ≤ |a| < 10

**Secant**: (1) Trigonometric function sec(θ) = 1/cos(θ) (2) Line intersecting curve at two points

**Second Derivative**: Derivative of derivative; f''(x) measures concavity and acceleration

**Sequence**: Ordered list {aₙ}; a₁, a₂, a₃, ...

**Series**: Sum of sequence; Σ aₙ

**Set**: Collection of distinct objects

**Set Theory**: Mathematical study of sets; foundation of mathematics

**Sigma (Σ)**: Summation notation; Σᵢ₌₁ⁿ aᵢ = a₁ + a₂ + ... + aₙ

**Significance Level**: Probability threshold for rejecting null hypothesis (typically 0.05)

**Similar**: (1) Same shape but different size (2) Related by similarity transformation

**Simplex**: Generalization of triangle to n dimensions

**Simpson's Rule**: Numerical integration using parabolic approximations

**Simulation**: Using computational model to study system behavior

**Sine**: Trigonometric function; sin(θ) = opposite/hypotenuse in right triangle

**Singular**: (1) Matrix with determinant 0 (not invertible) (2) Point where function not well-behaved

**Singular Value Decomposition (SVD)**: Matrix factorization A = UΣVᵀ; generalizes eigendecomposition

**Slope**: Steepness of line; rise over run; m in y = mx + b

**Solution**: Value(s) satisfying equation or system

**Span**: Set of all linear combinations of vectors

**Sparse**: Matrix/graph with mostly zero entries/few edges

**Spectral Theorem**: Symmetric matrix can be diagonalized by orthogonal matrix

**Sphere**: Set of points equidistant from center in 3D; x² + y² + z² = r²

**Square Matrix**: Matrix with equal number of rows and columns

**Square Root**: √x satisfies (√x)² = x

**Stability**: Property of equilibrium; whether perturbations grow or decay

**Standard Deviation**: Measure of spread; σ = √variance

**Standard Form**: Conventional way of writing expression

**Stationary Point**: Where derivative equals zero; f'(x) = 0

**Statistics**: Science of collecting, analyzing, interpreting data

**Stochastic**: Involving randomness or probability

**Subgroup**: Subset of group that is itself a group

**Subset**: A ⊆ B if every element of A is in B

**Subspace**: Subset of vector space that is itself a vector space

**Substitution**: Replacing variable with expression; u-substitution in integration

**Sufficient Condition**: A is sufficient for B if A ⇒ B (A guarantees B)

**Sum**: Result of addition; Σ notation for extended sums

**Summation**: Adding sequence of numbers; Σᵢ₌₁ⁿ aᵢ

**Supremum**: Least upper bound of set

**Surface**: Two-dimensional manifold in 3D space

**Surjection (Onto)**: Function where every element of codomain is output for some input

**Symmetric**: (1) Matrix: Aᵀ = A (2) Relation: a ~ b ⟹ b ~ a (3) Function: Even function

**Symmetric Difference**: A Δ B = (A - B) ∪ (B - A); elements in exactly one set

**System of Equations**: Multiple equations to be satisfied simultaneously

## T

**Tangent**: (1) Trigonometric function tan(θ) = sin(θ)/cos(θ) (2) Line touching curve at point

**Tangent Line**: Line touching curve at point with same slope; approximates curve locally

**Tautology**: Logical statement always true (e.g., P ∨ ¬P)

**Taylor Series**: Function representation as infinite polynomial centered at point

**Tensor**: Multidimensional array; generalizes scalars, vectors, matrices

**Term**: Part of expression separated by + or - ; element in sequence

**Theorem**: Statement proven true from axioms and prior results

**Topology**: Study of properties preserved under continuous deformations

**Topological Space**: Set with notion of "nearness" (open sets)

**Torus**: Donut-shaped surface; Cartesian product of two circles

**Trace**: Sum of diagonal entries of square matrix

**Transcendental**: Not algebraic; not root of polynomial with integer coefficients (e, π)

**Transformation**: Function mapping one space to another; change of coordinates or variables

**Transitive**: Relation where a ~ b and b ~ c implies a ~ c

**Translation**: Shifting by constant vector; f(x) → f(x - a)

**Transpose**: Flipping matrix across diagonal; rows ↔ columns; (A^T)ᵢⱼ = Aⱼᵢ

**Transversal**: Line intersecting two or more other lines

**Trapezoid**: Quadrilateral with one pair of parallel sides

**Trapezoidal Rule**: Numerical integration using trapezoid approximations

**Tree**: Connected graph with no cycles; n vertices ⟹ n-1 edges

**Triangle**: Three-sided polygon

**Triangle Inequality**: |a + b| ≤ |a| + |b|; distance axiom

**Trigonometry**: Study of triangles and periodic functions (sin, cos, tan)

**Trivial**: (1) Obvious or simple case (2) Zero solution to homogeneous equation

**Truth Table**: Table showing truth values of compound proposition for all input combinations

**Turing Machine**: Abstract computational model; defines computability

**Twin Prime Conjecture**: Infinitely many primes p where p+2 also prime (unproven)

## U

**Unbounded**: Having no finite upper or lower bound

**Uncountable**: Set too large to be listed; larger than natural numbers (e.g., real numbers)

**Undefined**: Not assigned a value; 1/0 is undefined

**Union**: A ∪ B = {x : x ∈ A or x ∈ B}; combined elements

**Unique**: Only one object satisfying property

**Unit**: Element with multiplicative inverse

**Unit Circle**: Circle with radius 1 centered at origin; x² + y² = 1

**Unit Vector**: Vector with length 1; direction indicator

**Universal Quantifier**: ∀ ("for all"); ∀x P(x) means P(x) true for every x

**Universal Set**: Set containing all objects under consideration

**Upper Bound**: Value greater than or equal to all elements in set

## V

**Variable**: Symbol representing unknown or changing quantity

**Variance**: Measure of spread; Var(X) = E[(X - μ)²] = σ²

**Vector**: Ordered list of numbers; represents magnitude and direction

**Vector Field**: Assignment of vector to each point in space

**Vector Space**: Set of vectors with addition and scalar multiplication satisfying axioms

**Velocity**: Rate of change of position; derivative of position function

**Venn Diagram**: Visual representation of set relationships using overlapping circles

**Vertex**: (1) Point where edges meet in graph (2) Corner of polygon (3) Maximum/minimum of parabola

**Volume**: Three-dimensional measure of space occupied

## W

**Wave Equation**: PDE modeling wave propagation; ∂²u/∂t² = c²∇²u

**Weighted**: Having associated numerical values; weighted average, weighted graph

**Well-Defined**: Unambiguous; each input gives unique output

**Work**: Force times distance; W = **F** · **d**

## X

**x-axis**: Horizontal axis in coordinate system

**x-intercept**: Where graph crosses x-axis; y = 0

## Y

**y-axis**: Vertical axis in coordinate system

**y-intercept**: Where graph crosses y-axis; x = 0; b in y = mx + b

## Z

**Zero**: Additive identity; 0 + a = a for all a

**Zero Divisor**: Non-zero element whose product with another non-zero element is zero

**Zero Matrix**: Matrix with all entries 0

**Zero Vector**: Additive identity in vector space; **0** + **v** = **v** for all **v**

**Zeta Function**: ζ(s) = Σ 1/nˢ; Riemann hypothesis concerns its zeros

- ℝ: Real numbers (all points on number line)

This glossary covers terminology across all mathematical subdisciplines studied in this guide. Terms appear in multiple contexts—understanding how concepts connect across branches reveals mathematics as unified whole rather than collection of isolated topics.
