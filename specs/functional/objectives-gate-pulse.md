# Objectives Gate Pulse

**Status:** Draft
**Author:** Raj
**Date:** 2026-07-02

## Problem

Weekly reviews open with narrative, not measurable reality. The user tracks
hard commitments (gates with targets, a go/no-go date, runway) outside the
app; the review should lead with those numbers so drift is visible.

## Users

A single user making high-stakes go/no-go decisions who maintains
`~/coach/objectives.yaml` by hand.

## Desired Behavior

1. The user maintains `~/coach/objectives.yaml`:

   ```yaml
   last_reviewed: 2026-07-02
   gates:
     - name: LP discovery conversations
       target: 15
       current: 4
   clock:
     runway_ends: 2028-06
     go_no_go: 2026-11-01
   ```

2. The weekly review leads with a **GATE PULSE** section rendered
   deterministically from this file: one line per gate (`current / target`),
   days to `go_no_go`, runway end. Numbers only — the section is prepended
   to the review output in code, not generated by the LLM, so no commentary
   can be produced about the numbers.
3. If the file is absent, the section is omitted silently.
4. If `last_reviewed` is more than 42 days old, one line is prepended:
   "objectives.yaml last reviewed {date} — stale measuring stick."
5. The advisor never writes this file. It is read-only user input.
6. A malformed file logs a warning and the section is omitted — the weekly
   review never crashes on it.

## Acceptance Criteria

- [ ] Valid file → review output begins with the GATE PULSE lines.
- [ ] Absent file → review output unchanged, no errors.
- [ ] `last_reviewed` 43+ days ago → staleness line prepended.
- [ ] Invalid YAML / schema → warning logged, review proceeds without pulse.
- [ ] Loader validates gates (name, numeric target/current) via Pydantic.

## Edge Cases

| Scenario | Expected Behavior |
|----------|-------------------|
| Empty file | Treated as absent (no pulse) |
| `clock` missing | Gates rendered, no clock lines |
| `go_no_go` in the past | Negative day count shown — the number speaks |
| Non-numeric target | Validation error → warning, pulse omitted |

## Out of Scope

- Any UI for editing objectives.
- Advisor commentary, trend analysis, or recommendations about gate values.
